//{Текст}
- переслать сообщение в чат от моего имени,
любой медиа кидаете мне, я в чат;

k/
- чтение файла ключей,
номер строки в нем соответсвует номеру файла, в котором хранятся фразы;

{№}/
- чтение файла с фразами;

{№}/{Текст}
- запись новой фразы в конец существующего файла;

{№}/{№}/{Текст}
- исправить фразу {№} в существующем файле;

{№}/0/{Текст}
- исправить ключ в существующем файле;

nf/{Ключ}
- создать ноый файл с новым ключом,
его можно будет начинять фразами
с помощью 	{№}/{Текст}
или 		{№}/{№}/{Текст}

l/
- посмотреть номера файлов;

t/{Минут}
- время ожидания активности в чате;

z/{Штук}
- количество пропущенных постов в режиме #заткнись;
